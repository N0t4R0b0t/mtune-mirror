# pkgmirror documentation

Detailed documentation for the pkgmirror multi-arch package mirror builder.
For a quick start, see the [top-level README](../README.md).

- **[Architecture](architecture.md)** — components, runtime layout, build & serving
  flow, parallelism, and the trust model, with diagrams.
- **[Data model](data-model.md)** — the configuration schema (arches, groups,
  packages, global settings), how the effective build set is derived, state files,
  and the repository layout, with diagrams.
- **[User guide](user-guide.md)** — installing, client setup, the web UI, the CLI,
  adding an architecture, patching packages, managing groups, pausing, and
  troubleshooting.
